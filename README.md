# Online Course Enrollment Dashboard 2024

## Project Overview
This Data Analytics project was developed using Microsoft Excel to analyze online course enrollment data and uncover meaningful insights through interactive dashboards and visualizations.

## Key Highlights
- Total Enrollment & Completion Analysis
- Course Dropout Rate Tracking
- Revenue Performance by Country
- Payment Status Analysis (Paid, Cancelled, Pending)
- Platform-wise Performance Comparison
- Course Category Distribution Insights

## Tools & Skills Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Modeling
- Dashboard Design

## Insights Generated
- Identified top-performing learning platform based on ratings
- Analyzed country-wise revenue contribution
- Evaluated course completion and dropout patterns
- Studied payment behavior trends across users
- Built interactive filters for dynamic data exploration

## Outcome
This project helped strengthen my skills in data analysis, dashboard creation, and turning raw educational data into actionable business insights using Excel.
